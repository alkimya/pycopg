  </what-built>

       it runs `uv lock --check`, `uv build`, then publishes via OIDC to the `pypi` environment.
    5. Confirm the release is live: https://pypi.org/project/pycopg/0.7.0/ resolves.
    Record the tag, the GitHub Release URL, and the workflow run URL in 29-03-RELEASE-LOG.md.
