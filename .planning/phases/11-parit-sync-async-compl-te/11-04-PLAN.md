| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-11-07 | Tampering (SQL injection) | async create/create_from_env/drop_extension identifiers | mitigate | Mirror sync `validate_identifier(name)`, owner, `validate_identifier(template)` / `validate_extension_name` before any f-string interpolation; existence check uses a `%s` placeholder, not an f-string. |
| T-11-08 | Elevation of Privilege | admin connect to `postgres` with autocommit | accept | Same posture as the established sync `create`/`create_database`; credentials come from the caller's own Config (no new secret source). DB creation inherently needs admin rights; not a new boundary. |
