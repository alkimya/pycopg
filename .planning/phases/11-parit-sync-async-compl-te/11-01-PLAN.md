<objective>
Fix C3 (PAR-06): the async SQLAlchemy engine is currently built from `config.url`, which uses the SYNC psycopg driver (`postgresql+psycopg://`). Add an explicit `Config.async_url` property that emits the async driver URL (`postgresql+psycopg_async://`) and rewire `AsyncDatabase.async_engine` to use it.

Purpose: A correct async driver URL is the foundation for every async DataFrame/engine operation (from_dataframe, to_dataframe, from_geodataframe). This plan runs in Wave 1 and is the first edit to async_database.py; all later async plans depend on it.
Output: `Config.async_url` property + rewired `async_engine`, plus a NO-DB unit test asserting the produced string (D-04).
</objective>
