<task type="auto">
  <name>Task 2: Add retry to connect methods, configurable batch size, and pool resilience params</name>
  <files>pycopg/database.py, pycopg/async_database.py, pycopg/pool.py</files>
  <action>
  **database.py — Retry on connect():**

  1. Add imports at the top:
     ```python
     import logging
     from tenacity import retry, stop_after_attempt, wait_exponential, retry_if_exception_type, before_sleep_log
     from psycopg import OperationalError
     ```
     Add logger: `logger = logging.getLogger(__name__)`

  2. Add a private method `_connect_with_retry` that wraps `psycopg.connect()` with tenacity retry:
     ```python
     @retry(
         stop=stop_after_attempt(3),
         wait=wait_exponential(multiplier=1, min=1, max=10),
         retry=retry_if_exception_type(OperationalError),
         before_sleep=before_sleep_log(logger, logging.WARNING),
         reraise=True,
     )
     def _connect_with_retry(self, autocommit: bool = False) -> psycopg.Connection:
         """Establish connection with retry for transient failures."""
         return psycopg.connect(**self.config.connect_params(), autocommit=autocommit)
     ```

  3. Update the existing `connect()` context manager to use `_connect_with_retry` instead of calling `psycopg.connect()` directly:
     Replace `conn = psycopg.connect(**self.config.connect_params(), autocommit=autocommit)` with `conn = self._connect_with_retry(autocommit=autocommit)`

  4. Update `insert_batch()` signature: change `batch_size: int = 1000` to `batch_size: Optional[int] = None`. At the start of the method body (after `if not rows: return 0`), add:
     ```python

     ```
