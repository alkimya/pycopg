4. **docs/async-database.md**: Add new sections after the existing "Schema Operations" section:

   ## DataFrame Operations (NEW in 0.3.0)
   Show: await db.to_dataframe("table"), await db.from_dataframe(df, "table"), await db.to_geodataframe("table"), await db.from_geodataframe(gdf, "table", spatial_index=True)
   Note: Uses run_sync internally since pandas/geopandas are synchronous libraries.

   ## Admin Operations (NEW in 0.3.0)
   Show: await db.create_table(name, columns_dict), await db.drop_table("name"), await db.create_index("table", "column", unique=True), await db.drop_index("name"), await db.list_indexes("table"), await db.list_constraints("table"), await db.drop_schema("name", cascade=True), await db.table_sizes("public")

   ## Maintenance Operations (NEW in 0.3.0)
   Show: await db.vacuum("table", analyze=True), await db.analyze("table"), await db.explain("SELECT ...", analyze=True)

   ## Backup & Restore Operations (NEW in 0.3.0)
   Show: await db.pg_dump("backup.dump"), await db.pg_restore("backup.dump", clean=True), await db.copy_to_csv("table", "file.csv"), await db.copy_from_csv("table", "file.csv")
   Note: pg_dump/pg_restore use asyncio.create_subprocess_exec internally.

   ## Database Lifecycle (NEW in 0.3.0)
   Show: await db.create_database("name", owner="user"), await db.drop_database("name")

   ## Role Management (NEW in 0.3.0)
   Show: await db.create_role("name", password="secret"), await db.drop_role("name"), await db.alter_role("name", password="new"), await db.grant("SELECT", "table", "role"), await db.revoke("SELECT", "table", "role"), await db.grant_role("role", "member"), await db.revoke_role("role", "member"), await db.list_role_members("role"), await db.list_role_grants("role")

   ## PostGIS Operations (NEW in 0.3.0)
   Show: await db.create_spatial_index("table"), await db.list_geometry_columns()

   ## TimescaleDB Operations (NEW in 0.3.0)
   Show: await db.create_hypertable("table", "time_column"), await db.enable_compression("table", segment_by="col"), await db.add_compression_policy("table", compress_after="7 days"), await db.add_retention_policy("table", drop_after="90 days"), await db.list_hypertables(), await db.hypertable_info("table")
